O que é HTML?

HTML é a sigla para HyperText Markup Language, que em português significa "Linguagem de Marcação
de Hipertexto" Mas calma! Isso não quer dizer que é algo difícil.

A palavra “linguagem de marcação” quer dizer que o HTML marca o conteúdo de um site. Ele organiza o
que vai ser título, o que vai ser parágrafo, onde vai uma imagem, um link, uma lista, e por aí vai.

Por exemplo, se você quiser escrever um texto e dizer que aquilo é um título, você vai usar uma tag
específica pra isso. Se for um parágrafo, outra tag. É como montar uma estrutura pra o que vai aparecer na página.

Como funciona o HTML?

Um arquivo HTML é como um esqueleto de uma página web. Ele não dá estilo, nem comportamento. Ele só organiza
o conteúdo. É como se você estivesse desenhando um mapa do que vai ter na tela.

Imagina uma folha em branco onde você coloca:

  • Um título lá em cima
  • Um parágrafo explicando alguma coisa
  • Uma imagem no meio
  • Um botão que leva pra outro site

Tudo isso pode ser feito com HTML.
