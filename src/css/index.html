O que é CSS?

CSS é a sigla para Cascading Style Sheets, que em português significa Folhas de Estilo em Cascata. Ele é uma linguagem
de estilo usada para definir a aparência de páginas feitas com HTML.

Com o CSS, podemos personalizar um site: escolher cores, fontes, tamanhos, alinhar textos, posicionar elementos, criar animações
e muito mais. Ele transforma uma página simples em algo visualmente agradável e bem organizado.

O que o CSS pode fazer?

Com o CSS voçê pode:

  • Mudar cores de textos e fundos.
  • Definir tamanhos de fontes e estilo de letras.
  • Adicionar espaçamentos e margens.
  • Criar bordas, sombras e transições.
  • Fazer layouts responsivos (que se adaptam ao celular ou computador).
  • Criar animações e efeitos visuais.
  • Organizar os elementos com Flexbox ou Grid.

O CSS é a alma visual da web. Com ele, você consegue transformar um site comum em algo bonito, organizado e funcional.
Junto com o HTML e o JavaScript, ele forma a base do desenvolvimento web.
